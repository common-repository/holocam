holocam.io wordpress plugin is a plugin to attach holocam.io panoramas in your wordpress site. 

== Description ==

How it works?
Get the zip file of your holocam.io panorama image from https://dashboard.holocam.io. Go to holocam.io Panorama Images plugin in admin menu.
Click “Add new” button, give a title as image name, and upload your zip in the attachment box. If upload succeed, you will see the image thumbnail
and a preview link to your 360 degree panorama on your webserver. Give a description, it will show up under your image thumbnail on frontend.
Set privacy if you want (only images with privacy: public will show up on frontend), default privacy is public.
Publish your holocam.io post and it is done.
To show the image on frontend use shortcodes on location of your choice.
Examples of how to use shortcodes  in holocam.io settings in the admin menu of your wordpress website.  

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/holocam` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the holocam.io settings screen to configure the plugin and entering your holocam.io API key.

== Frequently Asked Questions ==

= How can I get the API Key? =

If you have an account at https://dashboard.holocam.io, you can find it there.
